# Event Seeds: By Mode of Participation

> How do you engage with the event? What is demanded of you?

---

## Spectator

Just show up and receive. Presence without production.

### Listening Events
- **Concerts** — Orchestral, chamber, rock, jazz
- **Lectures** — Academic, public, TED
- **Sermons/talks** — Spiritual teaching
- **Podcasts live** — Live recordings
- **Audiobook readings** — Author performance

### Watching Events
- **Theater** — Drama, dance, opera
- **Film screenings** — Cinema, festivals, repertory
- **Sports events** — Games, matches
- **Art exhibitions** — Gallery, museum openings
- **Demonstrations** — Cooking, craft, technique

### Hybrid Spectating
- **Stand-up comedy** — Watching but responsive
- **Improv shows** — Audience participation optional
- **Magic shows** — Interactive moments
- **Debates** — Witnessing exchange
- **Q&A sessions** — Watching others ask

### Spectator Stances
- **Passive absorption** — Let it wash over
- **Critical watching** — Analyze as you receive
- **Devotional attendance** — Sacred witnessing
- **Social spectating** — Experience shared with others
- **Solo spectating** — Private reception

### The Value of Spectatorship
- **Low barrier** — No skill required
- **Discovery** — Exposure to new forms
- **Rest** — Receiving without producing
- **Inspiration** — Fuel for later creation
- **Community** — Shared attention creates bonds

## Participant

Active engagement but not central. Contributing to something larger.

### Movement Participation
- **Group exercise classes** — Yoga, aerobics, spin
- **Dance classes** — Partner, group
- **Martial arts practice** — Training sessions
- **Group hikes** — Walking together
- **Team sports** — Playing positions

### Discussion Participation
- **Book clubs** — Reading and discussing
- **Seminars** — Academic discussion
- **Workshops** — Skill-sharing events
- **Support groups** — Shared processing
- **Salons** — Curated conversation

### Making Participation
- **Group art projects** — Collaborative creation
- **Ensemble playing** — Orchestra, band, choir
- **Co-cooking** — Communal meal preparation
- **Hackathons** — Team building
- **Writing groups** — Parallel creation

### Ritual Participation
- **Congregational worship** — Shared liturgy
- **Group meditation** — Sitting together
- **Ceremonies** — Weddings, funerals as attendee
- **Festivals** — Active participation in activities
- **Parades/marches** — Moving with crowd

### Learning Participation
- **Class/course** — Student in cohort
- **Apprenticeship** — Learning by doing
- **Internship** — Working while learning
- **Mentorship** — Receiving guidance

### Participation Stances
- **Full engagement** — All in
- **Peripheral participation** — Legitimate but edge
- **Critical participation** — Engage while questioning
- **Experimental participation** — Trying on modes
- **Support participation** — Enabling others

## Presenter

You produce something to be witnessed. Central attention.

### Performance
- **Solo performance** — Concert, reading, lecture
- **Ensemble performance** — Band, theater, dance company
- **Improvised performance** — Jazz, improv comedy
- **Competitive performance** — Audition, competition
- **Ritual performance** — Officiating, leading

### Exhibition
- **Art show** — Gallery, museum exhibition
- **Portfolio review** — Showing work to experts
- **Demo day** — Startup pitches
- **Poster session** — Academic presentation
- **Open studio** — Process made visible

### Speaking
- **Keynote** — Featured address
- **Panel** — Shared discussion platform
- **Workshop leading** — Teaching through doing
- **Lecture** — Academic presentation
- **Sermon/talk** — Spiritual teaching

### Publication
- **Book launch** — Public release event
- **Album release** — Music publication event
- **Premiere** — Film/theater first showing
- **Product launch** — Startup/company announcement
- **Website launch** — Digital publication moment

### Presenter Stances
- **Authority** — Expert sharing knowledge
- **Witness** — Sharing experience
- **Provocateur** — Challenging audience
- **Host** — Creating container for others
- **Demonstrator** — Showing how

### Presentation Risks
- **Exposure** — Being seen fully
- **Judgment** — Being evaluated
- **Failure** — Public falling short
- **Success** — New expectations
- **Visibility** — Changed social position

## Organizer

You create the container. Making the event possible.

### Event Production
- **Conference organizing** — Large-scale coordination
- **Festival organizing** — Multi-venue, multi-day
- **Series/salon curation** — Ongoing program
- **Workshop hosting** — Single event facilitation
- **Party hosting** — Social gathering creation

### Space Creation
- **Venue management** — Ongoing space stewardship
- **Pop-up creation** — Temporary space activation
- **Online community hosting** — Digital space maintenance
- **Retreat leading** — Immersive environment creation

### Community Building
- **Collective founding** — Starting a group
- **Meetup organizing** — Regular gathering coordination
- **Coalition building** — Connecting groups
- **Scene seeding** — Cultural infrastructure creation

### Curriculum Design
- **Course creation** — Educational program design
- **Workshop development** — Skill-sharing design
- **Training program design** — Professional development
- **Retreat design** — Immersive experience creation

### Organizer Functions
- **Logistics** — Making it happen practically
- **Curation** — Selecting what/who is featured
- **Facilitation** — Guiding the process
- **Documentation** — Recording what happened
- **Fundraising** — Resourcing the event

### Organizer Stances
- **Invisible** — Event runs, you're unseen
- **Host** — Welcoming, connecting
- **Director** — Shaping what happens
- **Servant** — Supporting others' work
- **Architect** — Designing the structure

## Mode Transitions

How you might move through modes:

### Classic Path
1. **Spectator** → Attend many events
2. **Participant** → Join activities
3. **Presenter** → Share your work
4. **Organizer** → Create containers

### Circular Return
- Organizers benefit from spectating more
- Presenters benefit from returning to participation
- Participants benefit from occasional organizing

### Simultaneous Modes
- Organizing while presenting
- Participating while spectating
- Teaching while learning
