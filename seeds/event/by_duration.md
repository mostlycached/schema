# Event Seeds: By Duration

> Time-bounded occurrences. How long does the event hold you?

---

- **Author readings** — Book events
- **Tech talks** — Meetup presentations
- **Artist talks** — Gallery events
- **Sermons/dharma talks** — Spiritual teaching

### Readings / Performances
- **Poetry readings** — Literary community
- **Open mics** — Amateur performance
- **Comedy shows** — Stand-up, improv
- **Chamber concerts** — Small-scale music
- **Dance performances** — Movement arts

### Classes / Workshops (Single Session)
- **Drop-in yoga/meditation** — Embodied practice
- **Cooking classes** — Single skill
- **Craft workshops** — Make-and-take
- **Writing workshops** — Single session
- **Tech tutorials** — Hands-on learning

### Social Gatherings
- **Parties** — Social mixing
- **Meetups** — Interest-based gathering
- **Salons** — Curated conversation
- **Potlucks** — Food-centered gathering
- **Happy hours** — Work-adjacent social

### Ceremonies
- **Sunset rituals** — Daily marking
- **Grace before meals** — Brief sacred pause
- **Morning services** — Daily religious practice
- **Remembrance moments** — Brief honoring

## Duration Laddering

How events of different durations compound:

### Entry to Depth
1. **Hours** → Sample (talk, class, performance)
2. **Day** → Taste (workshop, day retreat)
3. **Weekend** → Commit (intensive, retreat)
4. **Week+** → Immerse (residency, program)
5. **Months** → Transform (long-term commitment)

### Rhythm Integration
- **Weekly hour** → Regular practice class
- **Monthly day** → Day-long retreat
- **Quarterly weekend** → Intensive workshop
- **Annual week** → Yearly residency

---

*Duration determines depth. Short events open doors; long events walk through them. Know what you're ready for.*
