# Material Seeds: By Function

> Physical objects, substances, tools. What they do.

---

## Instruments

Enable production. Tools, devices for making.

### Writing Instruments
- **Fountain pens** — Slowed, deliberate writing (Lamy, Pilot, TWSBI)
- **Typewriters** — Permanence, rhythm (Olivetti, Hermes)
- **Pencils** — Erasable, sketching (Blackwing, Palomino)
- **Brush pens** — Calligraphy, gesture (Pentel, Tombow)

### Musical Instruments
- **Strings** — Violin, cello, guitar, bass
- **Keys** — Piano, organ, synthesizer
- **Wind** — Flute, clarinet, saxophone
- **Percussion** — Drums, mallets, hand drums
- **Electronic** — Synths, samplers, controllers

### Craft Tools
- **Woodworking** — Chisels, planes, saws
- **Metalworking** — Hammers, tongs, anvils
- **Textile** — Needles, looms, scissors
- **Ceramics** — Wheels, kilns, tools
- **Leatherworking** — Awls, punches, stamps

### Digital Tools
- **Computers** — General purpose machines
- **Drawing tablets** — Digital mark-making
- **Audio interfaces** — Sound capture
- **Cameras** — Image capture
- **3D printers** — Material production
