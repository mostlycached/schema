## About

This repository contains research materials, computational workflows, and audiobook production tools for the Hyperframes project. The work spans:

- **Sociological & Phenomenological Worlds** ‚Äî A comprehensive framework analyzing 105+ contemporary and historical worlds
- **Attention Architecture** ‚Äî Frameworks for understanding meta-cognitive attention systems
- **Braitenberg Vehicles** ‚Äî Synthetic psychology simulations and diagrammatic research
- **Intervention Techniques** ‚Äî Methods for generating variations on worlds and narratives
- **Audiobook Production** ‚Äî Generative narrative variations with music and speech synthesis

## Local Development

To run the site locally:

```bash
cd docs
bundle install
bundle exec jekyll serve
```

Then open http://127.0.0.1:4000/ ‚Äî changes auto-reload.

*Requires Ruby 3.x. We recommend [rbenv](https://github.com/rbenv/rbenv).*
