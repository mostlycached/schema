*How ONE injected skill alters relationships with existing skills*

Alex Chen is doing typography. Existing disposition:

| # | Existing Skill | Description |
|---|----------------|-------------|
| T1 | **Kerning** | Micro-spacing between letters |
| T2 | **Visual hierarchy** | Size/weight encode importance |
| T3 | **Beauty appreciation** | Aesthetic judgment, taste |

| T5 | **Slow refinement** | Iterate patiently until right |

---

## INJECTION: "Control the Center" (from Chess)

*Controlling central squares gives positional advantage*

### Relationship to Each Existing Skill:

**T1: Kerning** 
- Relation: **TENSION**
- Why: Kerning is micro-local (between letters). "Control center" is macro-global (whole composition).
- Effect: Kerning may feel irrelevant - "why fuss over micro when macro matters?"
- Result: Kerning skill may atrophy OR transform into "local tight, global loose"

**T2: Visual hierarchy**
- Relation: **STRONG ALLIANCE**
- Why: Both care about what's most important. Hierarchy says "this matters most." Center says "put it here."
- Effect: Hierarchy becomes spatialized - importance = centrality
- Result: Hierarchy skill AMPLIFIED. But loses: peripheral importance, edge hierarchy

**T3: Beauty appreciation**
- Relation: **AMBIGUOUS TENSION**
- Why: Center-control is strategic, not aesthetic. But centered compositions CAN be beautiful.
- Effect: Beauty judgment becomes "is this balanced?" rather than "is this interesting?"
- Result: Beauty skill narrows - only appreciates centered beauty. Loses: asymmetric beauty, tension beauty, edge beauty

**T4: Gestalt grouping**
- Relation: **MODERATE ALLIANCE**
- Why: Both care about spatial relationships. Gestalt groups by proximity. Center organizes by position.
- Effect: Grouping now happens around center - radial organization
- Result: Gestalt skill becomes "center-outward" thinking. Loses: edge-to-edge relationships, diagonal tensions

**T5: Slow refinement**
- Relation: **NEUTRAL**
- Why: Center-control doesn't care about speed. You can slowly refine a centered composition.
- Effect: Minimal interaction
- Result: Skill unchanged

## Emergent Tensions (Lines of Flight?)

**Tension 1: Local vs Global**
- Kerning (local) now feels in conflict with Center-control (global)
- Potential line of flight: Abandon micro-detail OR integrate local into global system

**Tension 2: Asymmetric beauty is now "wrong"**
- Beauty skill trained to like asymmetry now CONFLICTS with center-control
- Potential line of flight: Suppress asymmetric taste OR rebel against center-control

**Tension 3: Edge elements feel neglected**
- Gestalt used to see all relationships. Now periphery is "weak zone."
- Potential line of flight: Develop "center-edge" dynamic OR fully commit to centrality

## What Creates a TRANSVERSAL Line of Flight?

A transversal line emerges when:
1. Multiple existing skills form an alliance AGAINST the injected skill
2. Or the injected skill creates irreconcilable tension with a core existing skill

**In this case:**
- If "Beauty appreciation" and "Kerning" ally against "Center-control" → possible line of flight toward "expressive asymmetric typography" (rejecting chess injection)
- If "Hierarchy" and "Gestalt" ally WITH "Center-control" → possible line of flight toward "information architecture" (accepting chess injection)

## CRITICAL: Molar → Molecular Decomposition

### The Problem

If Alex Chen only knows "chess" as a **molar block**, recombination is impossible.

```
"I know chess" (molar) + "I know typography" (molar) = ???
```

These are opaque blocks. No creative combination possible. Just two separate skills sitting side by side.

### The Requirement

For creative recombination, Alex must **molecularize** the skill - become aware of its internal components:

```
"Chess" (molar) → 
    - center-control (molecular)
    - sacrifice logic (molecular)
    - opponent modeling (molecular)
    - position evaluation (molecular)
    - ...
```

Only THEN can specific molecular components transfer:

```
center-control (from chess, molecular) 
    + 
visual hierarchy (from typography, molecular) 
    = 
COMPOSITION: spatial importance encoding
```

### What Enables Molecularization?

| Condition | Description |
|-----------|-------------|
| **Reflection** | Stopping to ask "what am I actually doing?" |
| **Teaching** | Explaining to another forces articulation |
| **Breakdown** | When skill fails, you see its parts |
| **Cross-domain encounter** | New context reveals hidden assumptions |
| **Analysis** | Deliberate study of the skill's mechanics |

### Without Molecularization

If Alex just "plays chess" and "does typography" without awareness of internal movements:
- Skills remain opaque molar blocks
- No intentional transfer possible
- Only accidental/unconscious spillover
- Recombination happens TO Alex, not BY Alex

### With Molecularization

When Alex has broken down both skills to molecular level:
- Specific components become available for recombination
- Intentional injection becomes possible
- Alex can ask: "What if I apply center-control to this layout?"
- Creative agency over the process

### Implication for Lines of Flight

**Molar assemblages** have fewer lines of flight - they're rigid, opaque.

**Molecular awareness** multiplies lines of flight - each component can combine with others.

The more Alex articulates the internal structure of skills, the more **degrees of freedom** for actualization.

---

## Summary: Conditions for Actualization

A virtual path is MORE LIKELY to actualize when:

1. **High intensive difference** - strong tension or alliance
2. **Singularity proximity** - encounter that forces resolution
3. **Molecular awareness** - components are available for recombination
4. **Alliance coherence** - multiple skills pointing same direction
5. **Irreconcilable tension** - forcing escape via line of flight

Without #3 (molecular awareness), Alex is SUBJECT TO recombination.
With #3, Alex can be AGENT OF recombination.
